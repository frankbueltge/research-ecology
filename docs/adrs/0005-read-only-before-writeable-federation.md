# ADR 0005 — Read-only adapters before writeable federation

**Status:** ACCEPTED — confirmed after audit, 2026-07-14.

## Context

As drafted — and the audit strengthened it: a complete real encounter
(`enc-2026-001-calibration-gap-travels`, see `docs/ENCOUNTER-INVENTORY.md`) is fully
reconstructable from public read-only material. No engine repo needs to change for
Phases 1–3.

## Decision

As drafted: audit → protocol kernel + fixtures from the real encounter → read-only adapters →
public vertical slice → stop for Frank's conceptual/visual review → only then manifests,
write flows, interventions.

## Consequences

As drafted. The stop condition is binding: a technically complete but dashboard-like result
is redesigned, not expanded.
